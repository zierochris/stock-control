# Stock Control Dashboard — PT. AT Indonesia

Real-time stock control system untuk 4 area produksi:
- **CORE** — Raw casting (47 parts)
- **ACE-1** — Finishing Line 1 (10 parts)
- **ACE-2** — Finishing Line 2 (10 parts)
- **EDP** — Electro-Deposition / Coating (6 parts)

## Akses Dashboard

👉 **[Buka Dashboard](./stock_control_master.html)**

## Fitur
- Input checkpoint setiap 2 jam per shift
- Alert otomatis: 🔴 Critical / 🟡 Warning / 🟢 OK
- Import data via Excel (.xlsx)
- Sync otomatis ke Google Sheets
- Laporan akhir shift (PNG → WhatsApp)
- Dark & Light mode
- Mobile friendly
